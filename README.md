# Aprendiendo-SpringBoot

Este repositorio sirve como guía, tutorial y bitácora de aprendizaje para el desarrollo de aplicaciones backend utilizando **Spring Boot 3.x** y **Java**.

## IDE
Para el desarrollo de este proyecto se utiliza **IntelliJ IDEA**

## Host
El backend se encuentra desplegado en **[Render](https://render.com/)**. 

Render es una plataforma en la nube (PaaS) que automatiza el despliegue de aplicaciones conectándose directamente a este repositorio de GitHub. Se ha seleccionado esta opción debido a que ofrece un plan gratuito que permite ejecutar la máquina virtual de Java (JVM) sin necesidad de introducir métodos de pago de entrada.
